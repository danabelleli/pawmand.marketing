# Pawmand Brand Style Guide

## Logo

- Wordmark: "Pawmand" in lowercase
- The "a" is replaced with a paw print icon
- Include @getpawmand somewhere on each post but placement can vary per post

## Colors

Full brand color palette:

#edf1fd → soft lavender white
#f2f5fc → light blue white
#e9ecf6 → muted periwinkle
#222a4e → dark navy (primary dark)
#6e7593 → muted slate blue
#7a93cb → medium periwinkle blue
#D0DAF1 → light periwinkle blue
#fed5b9 → warm peach
#654833 → warm brown

### Color usage philosophy:

These are your brand colors but you are not limited to using them in a rigid way. Mix and layer them creatively across posts. Variety in color usage keeps the feed feeling fresh and dynamic. Every post should feel like it belongs to Pawmand but no two posts need to look identical.

## Typography

- Maximum 2 font styles per post
- No single required font family
- Draw inspiration from the reference images provided in the inspiration folder
- Fonts should feel warm, modern, and professional
- Avoid overly corporate or sterile fonts
- Avoid overly playful or childish fonts
- Bold expressive headlines work well for high impact posts
- Clean readable fonts work well for informational posts

## Design Style

- Clean and minimal
- Soft rounded corners, no harsh edges
- Warm but professional
- Premium planner meets modern SaaS
- Versatile — not every post looks the same

## Decorative Elements

- Subtle paw print icons
- Soft organic blob shapes
- Gentle wave dividers
- Warm peach/cream accent shapes
- Subtle sparkle or star accents
- Never overwhelming — always subtle

## Photography Style

When using real dog photos:

- Clean light backgrounds preferred
- Dogs should look happy and energetic, no agressive or sad imagery
- Warm and inviting feeling
- Can use stock photos from Unsplash or Pexels

## Illustration Style

- Flat illustration style
- Friendly and approachable
- Consistent with color palette
- Cute but not childish

## Post Format

- Primary format: 4:5 vertical (1080x1350px)
- Always include @getPawmand on every post placement can vary — corner, bottom, anywhere that feels natural

## What to NEVER do

- Never use pure black (#000000)
- Never use harsh borders or dividers
- Never use more than 2 fonts per post
- Never use colors completely outside the brand palette
- Never make it look corporate or cold
- Never use aggressive or negative imagery
- Never use cluttered busy layouts
